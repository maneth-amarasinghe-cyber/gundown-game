# Gundown Web Game

A browser version of the original Gundown Python command-line game.

## Run locally
Open `index.html` in a browser.

## Deploy to Vercel
This is a static HTML/CSS/JavaScript project. Upload the contents of this folder to a GitHub repository, import the repository into Vercel, and deploy with no build command.

## Original rules retained
- Grid sizes from 3x3 to 5x5
- Hidden targets are between 2 and half of the total cells
- Hidden positions are random
- Players choose unique cells
- Results show found and missed targets
- Final percentage is calculated
- Results can be saved as a `.txt` file
